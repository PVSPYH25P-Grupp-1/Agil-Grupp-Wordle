Här kommer vi att samla all info om logik och regler för appen. 

Detta gäller i vårt spel:
En "spelare" kan enbart spela en gång (per spelomgång dvs per körning av appen)
Det är bara engelska ord
Det finns ord som inte får användas, som är olämpliga
Vilka ord som går att spela med styrs genom ordlista i appen
Det krävs inget login - det innebär att man kan starta om appen och det räknas då som ny spelomgång, dvs ordet är nytt
Man får 6 gissningar per spelomgång
Om man gissar rätt inom de gissningarna vinner man
Om man inte gissar på rätt ord inom 6 försök förlorar man
